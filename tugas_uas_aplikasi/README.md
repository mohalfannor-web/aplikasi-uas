# TaskMate

## Deskripsi

TaskMate adalah aplikasi mobile yang membantu mahasiswa mengatur tugas dan jadwal kuliah agar lebih terorganisir.

Aplikasi ini dibuat menggunakan Flutter dan dapat berjalan secara offline.

---

## Fitur Aplikasi

1. Dashboard
   Menampilkan ringkasan tugas dan deadline.

2. Manajemen Tugas
   User dapat menambahkan, mengedit, dan menghapus tugas.

3. Jadwal Kuliah
   Menyimpan jadwal mata kuliah.

4. Statistik
   Menampilkan jumlah tugas selesai dan belum selesai.

---

## Teknologi yang Digunakan

- Flutter
- Dart
- SQLite / Hive
- Git

---

## Cara Menjalankan Aplikasi

1. Clone repository

git clone https://github.com/username/studymate.git

2. Masuk ke folder project

cd studymate

3. Install dependencies

flutter pub get

4. Jalankan aplikasi

flutter run
